# Backend

This folder contains the Node.js + Express API for the Flutter app.

## What it does

- Stores user profile data in MySQL
- Exposes CRUD endpoints for Flutter
- Uses a single `users` table with JSON columns for list fields

## Next step

1. Copy `.env.example` to `.env`
2. Create the MySQL database
3. Run the SQL in `sql/schema.sql`
4. Install dependencies with `npm install`
5. Start the API with `npm run dev`
